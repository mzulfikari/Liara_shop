## 📌 توضیحات پروژه
پروژه LiaraShop یک فروشگاه آنلاین کامل و کاربرپسند است که با استفاده از فریم‌ورک Django در  توسعه یافته است. این پروژه امکاناتی همچون مدیریت محصولات، سبد خرید، حساب کاربری، و تجربه خرید ساده و امن را در اختیار کاربران قرار می‌دهد،کنترل خطا ها اضافه کردن آدرس های پیش فرض و یک داشبورد برای مشتری امکان کاربری راحت را فراهم می کند تا به کاربر امان صحولت کار را بدهد.
